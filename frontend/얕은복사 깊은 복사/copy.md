# 얕은복사 깊은 복사에 대해 알아봅시다!!

## 먼저 **얕은 복사**에 대해 알아봅시다

```
const obj = {name: "강현"};
const objCopy = obj;
```

여기서 objCopy에는 무엇이 들어갈까요?

처음 보는 사람은
{ name: "강현" } 객체 자체가 그대로 복사돼서,
"같은 내용을 가진 객체 두 개"가 생긴다고 생각할 수 있습니다.

하지만 그게 아닙니다.

objCopy에는 객체 그 자체가 아니라,
{ name: "강현" } 이 저장돼 있는 메모리 주소(참조값) 이 들어갑니다.
즉, 두 변수는 같은 객체를 가리키게 됩니다.

그럼 이 코드는 무엇이 출력이 될까요?

```
const obj = {name: "강현"};
const objCopy = obj;

objCopy.name = "연돈";
console.log(obj)
```

"연돈"이 출력이 됩니다.

왜 그럴까요?

이 코드는 객체 내부의 값을 새로 가져오는 것이 아니라,
obj가 가리키는 메모리 주소를 그대로 공유하는 것이기 때문입니다.
그래서 objCopy에서 값을 변경하면
obj 또한 같은 주소를 바라보고 있기 때문에 같이 바뀌게 됩니다.

이런 방식을 **얕은 복사**라고 합니다.

## 그럼 **깊은 복사**는 무엇일까요?

깊은 복사는 말 그대로
객체 내부의 값들을 실제로 새롭게 복사해서 완전히 독립된 객체를 만드는 것입니다.

즉, 서로 다른 주소를 갖기 때문에
한쪽을 바꿔도 다른쪽은 전혀 영향을 받지 않습니다.

## **요약**

얕은 복사는
주소값을 참조하는거고 <br>
깊은 복사는
실제 내부의 값을 복사해서 가져오는거임 ㅇㅇ
